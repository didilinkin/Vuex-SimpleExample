# vuex-simple Example

> vuex-simple Example:Vue components + Vuex,Counter App;
> 
> 一个最简单的Vuex示例: 组件与Vuex 管理状态, 一个计数器例子

:smirk: Haha

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
